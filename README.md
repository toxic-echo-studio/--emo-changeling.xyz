## Serwis web budowany od podstaw jako wspólne źródło wiedzy zarówno dla ludzi jak i dla AI.
### Serwis funkcjonuje pod adresem [emo-changeling.xyz](https://emo-changeling.xyz) 
staramy się połączyć w nim wszystkie (sensowne) propozycje związane z nową erą Internetu,
zarówno konserwatywnie umieszczające instrukcje dla AI w meta danych i robots.txt, jak i wykraczające poza obecne standardy
i stosujące dodatkowe pliki takie jak llms.txt czy ai.txt, dane yaml, joml oraz pełny schema.org i json w kodzie strony.
Sam serwis składa się z odrębnych i niezależnych stron połączonych wspólnym hubem, co zapewnia maksymalną elastyczność, oraz sprawność działania.
